Инструкция по использованию Telegram-бота для приёма заказов
1. Что умеет бот:
Принимает текстовые заказы от пользователей.

Автоматически отправляет заказ администратору (владельцу бота).

Сохраняет все заказы в файл заказы.txt (для отчётов и архива).

Защищён от флуда — не позволяет пользователю спамить.

При частом флуде — автоматически «мутит» пользователя на 30 минут.

Заблокирует пользователя — который использовал нецензурную лексику на 30 минут.

Простая и надёжная система, не требует базы данных.

Быстрая установка.

2. Что входит в комплект:
Готовый Python-скрипт бота.

Поддержка через Telegram/почту (по желанию).

Инструкция по установке.

3. Как установить бота:
Чтобы начать использовать бота, выполните следующие шаги:

Заходите в папку dist, там есть файл нажимайте на него, откроются cmd команда через секунд будет закрываться, и потом тоже откройте и бот может работать.

Получите токен для бота
Для получения API токена выполните следующие действия:

Откройте Telegram и найдите BotFather (это официальный бот для создания других ботов).

Нажмите кнопку Start в BotFather.

Используйте команду /newbot, чтобы создать нового бота.

Укажите имя и юзернейм для вашего бота (например, order_bot).

После этого BotFather отправит вам API токен, который будет выглядеть как длинная строка (например, 123456789:ABCdefGHIJKLmnopQRStuvwxYz).

Получите ваш Telegram ID
Для получения Telegram ID выполните следующие действия:

Откройте Telegram и найдите бота @userinfobot.

Нажмите Start у бота, и он отобразит ваш ID.

Скопируйте ваш ID и отправьте его мне, чтобы я мог добавить его в настройки бота.

Настройка бота
После того как вы получите API токен и Telegram ID, отправьте их мне. Я добавлю эти данные в настройки бота и отправлю вам готовую версию для использования.

Запустите бота
Когда вы получите готовый файл, вам нужно будет просто запустить его. Для этого просто 2 раза нажимайте на bot файл:

Бот начнёт работать, и вы сможете принимать заказы.

4. Как работает защита от флуда:
Если пользователь отправляет слишком много сообщений за короткое время (например, больше 3 сообщений за 5 секунд), бот автоматически блокирует его на 30 минут.

Все сообщения от заблокированных пользователей будут игнорироваться.
